# Fenchel_Duality_of_Nonlinear_Eigenproblems_Computation_of_Eigenfunctions
Implementation and examples of the numerical examples in the paper Fenchel Duality of Nonlinear Eigenfunctions

For an easy way to use the code you can create a virtual environment using conda with the environment.yml file.

`conda env create -f environment.yml`

See https://docs.conda.io/projects/conda/en/latest/user-guide/tasks/manage-environments.html for details.
